BinDot2Braille
==============

You can convert from

    00010000
    00010000
    11111110
    00111000
    00111000
    01101100
    01000100
    00000000

to

    ⠤⣼⡤⠄
    ⠰⠋⠳⠀

== text2braille

and you can use ASCII interactive converter shell.

    $ text2braille
    YAPPY> hi
    ⣆⠀⢐⠀
    ⠇⠇⠸⠀
    YAPPY> YAPPY
    ⢆⠆⡔⡄⡖⡄⡖⡄⢆⠆
    ⠸⠀⠏⠇⠏⠀⠏⠀⠸⠀
    YAPPY> BAD BOY
    ⣖⠄⡔⡄⡖⡄⠀⠀⣖⠄⡔⡄⢆⠆
    ⠧⠃⠏⠇⠧⠃⠀⠀⠧⠃⠣⠃⠸⠀
    YAPPY> I have a pen.
    ⢲⠂⠀⠀⣆⠀⢀⡀⡀⡀⢀⡀⠀⠀⢀⡀⠀⠀⣀⠀⢀⡀⣀⠀⠀⠀
    ⠼⠄⠀⠀⠇⠇⠣⠇⠗⠁⠫⠅⠀⠀⠣⠇⠀⠀⡧⠃⠫⠅⠇⠇⠠⠀
    YAPPY> quit
